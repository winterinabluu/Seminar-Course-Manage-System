# Seminar-Course-Manage-System

> 说明：这是早期学习 Java Web 和课程项目开发时的练习作品，仅用作留档记录。

`Seminar-Course-Manage-System` 是一个研讨课/课程管理系统练习项目，后端使用 Spring Boot、MyBatis、MySQL 等技术，前端包含面向教师和学生的页面。项目主要围绕课程、班级、讨论课、组队、话题和报告等教学场景展开。

项目创建提交日期：2018-12-29。这个仓库更像一次课程实践的完整留档，适合回看当时如何把后端接口、数据库和前端页面串起来。

## 功能概览

- 教师端课程、班级、讨论课和话题管理页面。
- 学生端选课、组队、讨论课参与和报告相关页面。
- 后端使用 Spring Boot 提供业务逻辑和数据访问。
- 使用 MyBatis、MySQL 处理课程管理相关数据。
- 包含部分前端静态页面、样式和脚本资源。

## 运行环境

- Java 8。
- Maven。
- Spring Boot 2.0.x。
- MySQL。

## 打开与运行

1. 准备本地 MySQL 数据库，并根据项目配置调整数据库连接信息。
2. 使用 IntelliJ IDEA 或其他 Java IDE 导入 Maven 项目。
3. 安装依赖后运行 Spring Boot 启动类。
4. 通过浏览器访问对应页面进行测试。

## 开发小结

- 这个项目练习了从需求页面到后端接口、再到数据库表的完整开发链路。
- 教师和学生两套角色流程让业务状态变得更复杂，也更适合练习系统拆分。
- 现在回头看，项目结构和配置还有不少早期痕迹，但它很真实地记录了一次 Java Web 课程实践。
